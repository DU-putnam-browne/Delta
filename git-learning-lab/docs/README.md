# Documentation Index

Use these docs to run, teach, maintain, and release the Git Learning Lab.

| File | Use it when |
| --- | --- |
| `LEARNER_QUICKSTART.md` | A learner needs the fastest path into the lab |
| `FACILITATOR_GUIDE.md` | Someone is leading a live orientation or working session |
| `COMMAND_REFERENCE.md` | A learner wants the supported simulator command list |
| `REPO_SETUP.md` | The folder needs to become or move to a standalone GitHub repo |
| `TROUBLESHOOTING.md` | The app does not open or local state behaves oddly |
| `RELEASE_CHECKLIST.md` | The repo is being prepared for sharing or publishing |

## Operating Principle

Keep this repo boring in the best possible way: static files, clear docs, no required install, and enough validation to prevent broken links or missing assets.
