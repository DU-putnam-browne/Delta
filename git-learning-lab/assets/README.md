## Rules

- Keep file names stable unless `index.html` is updated in the same change.
- Do not use externally hosted images; this app should run offline after download.
- Compress large assets before committing.
- Avoid placing screenshots here. Use a future `references/` or `docs/images/` folder if release screenshots are needed.
